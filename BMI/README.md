# Dataset: BMI Dataset

This dataset contains information related to Body Mass Index (BMI) prediction, used for training and evaluating a K-Nearest Neighbors (KNN) classifier.

## Features:
- `Gender`: The gender of the individual (Male/Female).
- `Height`: The height of the individual in centimeters (cm).
- `Weight`: The weight of the individual in kilograms (kg).

## Target Variable:
- `Index`: A numerical index representing the BMI category. This dataset uses a 6-level scale (0-5) to classify BMI, ranging from 'Extremely Weak' to 'Extreme Obesity'.

## Source:
This dataset is sourced from Kaggle:
[BMI Dataset on Kaggle](https://www.kaggle.com/datasets/yasserh/bmidataset)
